

# 👨‍💻 Personal Portfolio Website

A personal portfolio website showcasing my skills, projects, and experiences as a full-stack developer. Built with React and Tailwind CSS for a clean, responsive design.

## 🧱 Tech Stack

- ⚛️ React.js (with Vite)  
- 🎨 Tailwind CSS  
- ⚙️ Vite as build tool  
- 🛠 ESLint, Prettier for code quality  
- 📄 TypeScript  

## ✨ Features

- Responsive layout for mobile and desktop  
- Modern UI with clean typography and animations  
- Project showcase with links to live demos and GitHub repos  
- Contact section for easy connection

## 📁 Project Structure

```bash
Portfolio/
├── public/                  # Static files like favicon, images  
├── src/
│   ├── components/          # Reusable UI components  
│   ├── pages/               # Page components  
│   ├── App.tsx              # Root React component  
│   └── main.tsx             # Entry point  
├── .eslintrc.cjs            # ESLint config  
├── .gitignore  
├── package.json             # Dependencies & scripts  
├── postcss.config.js        # PostCSS config  
├── tailwind.config.js       # Tailwind CSS config  
├── tsconfig.json            # TypeScript config  
├── vite.config.ts           # Vite config  
└── README.md                # This file  
```

🚀 Getti  
```bash
# 1. Clone the repo
git clone https://github.com/Foam-01/Portfolio.git

# 2. Move to project folder
cd Portfolio

# 3. Install dependencies
npm install

# 4. Start development server
npm run dev
```
